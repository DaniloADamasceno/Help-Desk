Olá, seja bem-vindo ao Help Desk

Este é um sistema para gerenciamento de chamados simulando um helpdesk, desenvolvido em Angular 12 e Spring Boot 2.

O projeto visa abordar os conceitos de desenvolvimento de uma aplicação web, utilizando o framework Angular 12 e Spring Boot 2.

O sistema possui as seguintes funcionalidades:

  • Autenticação de autorização via Tokens JWT
  • Aplicações SPAs
  • CRUDs diversos de: Usuários, Clientes, Técnicos, Chamados, Perfis de Acesso

Fique à vontade para testar o sistema, criar usuários, clientes, técnicos e chamados.
